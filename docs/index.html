📋 Collecta

Schema-driven field data collection — offline-first mobile forms, validation, attachments, and sync.

Features

📝 Form Schemas

20+ field types with conditional logic, validation constraints, and XLSForm-compatible structure.

📡 Offline-First Sync

Queue submissions offline with exponential backoff retry. Never lose field data.

📎 Attachments

Photos, audio, video, signatures, barcodes — up to 50MB per attachment with content-addressable storage.

🌍 GPS Capture

Point, trace, and shape geometry capture integrated into form fields.

✅ Validation

Required fields, range checks, regex patterns, unique constraints, conditional requirements.

🔌 REST API

Axum-powered server for form CRUD, submission ingestion, and sync status management.
